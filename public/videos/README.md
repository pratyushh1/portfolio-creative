# Videos Folder

Place all your video files here for the video editing portfolio section.

## Recommended Structure:
```
videos/
├── tech-burner/
│   ├── video-1.mp4
│   ├── video-2.mp4
│   └── thumbnails/
├── open-letter/
│   ├── video-1.mp4
│   └── thumbnails/
└── other-projects/
    └── ...
```

## Supported Formats:
- MP4 (recommended)
- WebM
- MOV

## Best Practices:
- Keep file sizes optimized (compress for web)
- Use descriptive filenames
- Include thumbnails for faster loading
- Recommended resolution: 1920x1080 or 1280x720
