# Designs Folder

Place all your graphic design portfolio work here.

## Recommended Structure:
```
designs/
├── branding/
│   ├── logo-1.png
│   ├── brand-identity-1.jpg
│   └── ...
├── posters/
│   ├── poster-1.jpg
│   └── ...
├── social-media/
│   ├── instagram-post-1.jpg
│   ├── youtube-thumbnail-1.jpg
│   └── ...
├── packaging/
│   └── ...
└── ui-ux/
    └── ...
```

## Supported Formats:
- PNG (recommended for designs with transparency)
- JPG (for mockups and presentations)
- SVG (for logos and vector graphics)
- WebP (best for web performance)

## Best Practices:
- Show final designs in context (mockups)
- Include before/after or process shots if relevant
- Optimize for web viewing
- Use descriptive filenames
- Recommended width: 1920px max
