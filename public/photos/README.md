# Photos Folder

Place all your photography portfolio images here.

## Recommended Structure:
```
photos/
├── portraits/
│   ├── portrait-1.jpg
│   ├── portrait-2.jpg
│   └── ...
├── products/
│   ├── product-1.jpg
│   └── ...
├── landscapes/
│   ├── landscape-1.jpg
│   └── ...
└── events/
    ├── event-1.jpg
    └── ...
```

## Supported Formats:
- JPG/JPEG (recommended for photos)
- PNG
- WebP (best for web performance)

## Best Practices:
- Optimize images for web (compress without losing quality)
- Recommended width: 1920px max
- Use consistent aspect ratios within categories
- Use descriptive filenames
